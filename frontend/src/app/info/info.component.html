CINEMIXER

How to use the Graph

Discover movies by mixing and matching elements.

  • Add Nodes: Type any word or phrase in the search bar to add it to the graph.
  • Create Hubs: Click on an element node to create a "hub". Mix multiple elements by adding them to the hub.
  • Generate Movies: Once you have elements in your hub, it will find movies matching your mix!
  • Interact: Drag nodes around, right-click for more options like adding weight or removing nodes.
